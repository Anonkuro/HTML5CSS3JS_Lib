HTML5

HTML5历史

HTML5 草案的前身名为 Web Applications 1.0， 于 2004年被 WHATWG 提出，于 2007 年被 W3C接纳，并成立了新的 HTML 工作团队。HTML5 的第一份正式草案已于 2008年1月22日公布。 2014年10月28日， W3C 的 HTML工作组正式发布了 HTML 的官方推荐标准。

名词解释

WHATWG
Web Hypertext Application Technology Working Group, HTML 工作开发组的简称，目前与 W3C 组织同时研发HTML5。
W3C
World Wide Web Consortium，万维网联盟，万维网联盟是国际著名的标准化组织。1994年成立后，至今已发布近百项相关万维网的标准，对万维网做出了杰出的贡献。
